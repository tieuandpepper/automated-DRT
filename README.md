## USE INSTRUCTION ##
1. Install `galvani` and `pandas` package from Python
    * Pip
        * pip install galvani
        * pip install pandas
    * Conda
        * conda install galvani
        * conda install pandas

2. Install `Optimization Toolbox` in MatLab

3. Matlab App
    * Open and import the .txt file with no header
    * Press simple run
    * Export
